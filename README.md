# Super Vandal World

This game is a 2D platformer, inspired by the University of Idaho mascot (Joe Vandal) and his adventures through difficulty obstacles.

Controls: </br>
* Jump -> Space </br>
* Power Up -> J </br>
* Left -> A </br>
* Right -> D </br>
